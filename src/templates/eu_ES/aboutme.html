Niri buruz

Gaur egun ingeniari informatikoa naiz eta Elecnor enpresan egiten dut lan, ingeniaritza elektrikoko eta energiako proiektuak garatzen dituen konpainia batean; nire lana mota askotako proiektuetan aplikazio eta sistema informatikoak garatzea da.

Uni Eibar-Ermua institutuan batxilergoa amaitu ondoren, informatikako ikasketak hasi nituen. Lehenik eta behin, software garapenean zentratutako goi-mailako lanbide heziketako zikloa egin nuen institutuan bertan. Ondoren, Ingeniaritza Informatikoko Gradua egin nuen, Software Ingeniaritzan espezializatuta. Unibertsitate berean, Sistema Informatiko Aurreratuak masterra osatu nuen. Azkenik, Euskal Herriko Unibertsitatean (UPV/EHU) doktoretza bukatu nuen Informatika Fakultatean. Nire ikerketa eremua web-eko anotazioak hainbat domeinutan aplikatzean oinarritzen da.

Ikasketak egin bitartean bost enpresatan aritu naiz web teknologiei lotutako lanetan, front-end zein back-end; hala ere, erabiltzailearen interfazearekin zerikusia dutenak gustatzen zaizkit gehien. Ez naiz Javascript, HTML edo CSS hizkuntzetan aditua, baina gehien erabiltzen ditudan tresnak dira. Adibidez, Graduko, Masterreko eta Doktorego tesiek Javascript hutsean daude garatuta.
